# badmoviesquad
Manage proceedings for Bad Movie Squad

## HOW TO RUN
```
yarn install
yarn start
```

### installing the latest LTS version of NodeJS
```
nvm install node --latest-npm --reinstall-packages-from=node
```

### installing node 10
```
nvm install 10.15.3
```

### to deploy
```
yarn build
firebase deploy
```
